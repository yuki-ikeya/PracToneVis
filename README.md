# PracToneVis
<p style="font-weight:bold; font-size:18px;">
入力音声 → 特徴抽出 → 可視化 → 気づき
</p>

🎥 Demo  
https://yuki-ikeya.github.io/PracToneVis/

管楽器奏者の演奏を“評価”ではなく“可視化”し、自己学習を支援するシステム
音量・音程・音色などの音響特徴を五線譜上に統合的に表示し、奏者自身が演奏の特徴やばらつきに気づけることを目指しました。
<p style="color:#64748b;">

<p>
従来の「評価」ではなく、可視化によって自己理解を促す点に新規性があります。
</p>

※現在、楽曲フレーズへの拡張を開発中
</p>
